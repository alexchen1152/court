METHODOLOGY
4Discovery utilized EnCase Forensic v8.12 ("Encase") to analyze the Windows Registry
extracted from the Oder Laptop. Below is a description of the artifacts reviewed to complete
this report. Consistent with digital forensic best practices, 4Discovery verified its findings using
the Windows Registry navigation tool, RegRipper.
USB Storage Devices
To identify evidence of USB storage devices connected to the Oder Laptop, 4Discovery
reviewed the following Windows Registry artifacts.
USBSTOR
The USBSTOR key is created by the system upon the enumeration of a USB device, which
generally occurs when a USB device is connected to the system. The USBSTOR key contains
information about USB storage devices that have been connected to the computer.
The USBSTOR key is located in the following location in the SYSTEM hive:
•	 SYSTEM\ControlSet00x\Enum\USBSTOR3
The USBSTOR key contains subkeys that are typically formatted as follows:
Disk&Ven_[VENDOR]&Prod_[PRODUCT]&Rev_[REVISION]
[SERIAL_NUMBER]
The [SERIAL_NUMBER] subkey is unique to each USB storage device. If a USB
storage device does not have a serial number, Windows will automatically create a unique
serial number.
3 "ControlSet00x" refers to the multiple Control Sets that exist within the Windows Registry, where "x" is a numeral.
Forensic tools often examine multiple Control Sets (e.g., ControlSet001, ControlSet002) to ensure comprehensive
analysis, as the active control set can vary.

Mounted Devices
The MountedDevices key is a registry key that contains information about all mounted
devices, such as USB drives, CD/DVD drives, and network shares. It is located under the
following registry path:
•	 SYSTEM\MountedDevices
The MountedDevices key contains a list of all mounted devices, along with their associated
drive letters and volume GUIDs. Each entry in the key has a value name that represents the
drive letter or volume GUID, and a value data that contains a binary structure with device
information.
When a USB storage device is connected, the operating system creates an entry in the
MountedDevices key. This entry is used to track volume information and the assigned drive
letter for future use.
DeviceClasses
The DeviceClasses key is a registry key that stores information about all device classes
installed on the computer. It is located under the following registry path:
•	 SYSTEM\CurrentControlSet\Control\DeviceClasses
Each device class is represented by a GUID-named subkey. The subkeys contain information
about each device that belongs to that class.
When a USB storage device is connected to the system, a new entry is created under the
corresponding device class subkey. One of the tracked device classes is portable devices, which
is listed under the GUID of {884b96c3-56ef-11d1-bc8c-00a0c91405dd}. When a new USB
storage device is connected, subkeys are created relating to the specific USB device and
tracking when the device was connected.

Anti-Forensic and/or Wiping Tools
To locate evidence of anti-forensic and/or wiping tools, 4Discovery analyzed the following
the Windows Registry artifacts.
ShimCache
The ShimCache, also known as Application Compatibility Cache, is a component of the
Windows operating system designed to identify application compatibility issues. ShimCache
stores various metadata about files, including the full file path, file size, last modified time, and
last execution time.
The ShimCache entries are located in the following locations in the SYSTEM hive:
•	 SYSTEM\ControlSet00x\Control\Session Manager\AppCompatCache
4Discovery reviewed the ShimCache entries to determine whether there was
evidence of the execution of wiping and/or anti-forensic tools, such as CCleaner, Eraser,
and BleachBit.

UserAssist
The UserAssist key is a registry key that records information about programs executed by a
specific user through the Windows graphical user interface (GUI). It tracks data such as the
name of the executable, the full file path, the number of times the program was run, and the
last execution time. The UserAssist key is located under the following registry path:
•	 SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\UserAssist
The UserAssist key contains subkeys, each identified by a GUID. Within each GUID subkey,
there are further subkeys, including a "Count" subkey. This "Count" subkey stores the execution
information.
4Discovery reviewed the UserAssist key entries to determine whether there was
evidence of the execution of wiping and/or anti-forensic tools, such as CCleaner, Eraser, and
BleachBit.
MUICache
The MUICache (Multilingual User Interface Cache) is a registry key that stores a list of
applications executed on the system. The MUICache entry contains information about executable
names, which may include the file path. The MUICache key is located under the following
registry path:
•	 SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\MuiCache
4Discovery reviewed the MUICache key entries to determine whether there was
evidence of the execution of wiping and/or anti-forensic tools, such as CCleaner, Eraser, and
BleachBit.

Installed Programs

Production Files
To locate evidence of files Fiserv requested for production (the "Production Files"),
4Discovery analyzed UserAssist, ShimCache, and MUICache entries, described above.
Additionally, 4Discovery analyzed the following Windows Registry artifacts.
OpenSavePidlMRU
The OpenSavePidlMRU (Most Recently Used) key is a registry key that stores a list of files
that have been opened or saved using the common file dialog box. This key tracks the
filenames, paths, and extensions of the files accessed through this dialog. The
OpenSavePidlMRU key is located under the following registry path:
•	 SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\ComDlg32\OpenSavePidlMRU
Within the OpenSavePidlMRU key, subkeys are organized by file type, such as "docx" or
"pdf," and each subkey contains a list of recently opened or saved files of that type ordered
by a numerical value. 4Discovery reviewed the OpenSavePidlMRU to determine whether
there was evidence of the Production Files.

LastVisitedPidlMRU
Similar to the OpenSavePidlMRU, the LastVisitedPidlMRU key is a registry key that stores
a list of the last locations accessed by the user through the common file dialog box. However,
the LastVisitedPidlMRU key keeps a record of the locations used to open files and the
locations used to save files. The LastVisitedPidlMRU key is located under the following
registry path:
•	 SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\ComDlg32\LastVisitedPidlMRU
4Discovery reviewed the LastVisitedPidlMRU to determine whether there was
evidence of the Production Files being opened or saved.
RunMRU
The RunMRU key is a registry key that maintains a list of commands executed via the "Run"
dialog box. This key tracks the commands entered by the user, which could include executable
files, scripts, or system commands. The RunMRU key is located under the following registry path:
•	 SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\RunMRU
Each command is stored as a string value with an associated letter, and an additional
"MRUList" value specifies the order in which the commands were executed. 4Discovery
reviewed the RunMRU to determine whether there was evidence of the Production Files.
RecentDocs
The RecentDocs key is a registry key that keeps track of recently opened documents. The
RecentDocs key is organized by file extension and contains MRU entries, keeping a record of
the files that have been opened by a particular user. The RecentDocs key is located under the
following registry path:
•	 SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\RecentDocs
4Discovery reviewed the RecentDocs to determine whether there was evidence of the
Production Files.

ShellBags
The ShellBags keys are a set of registry keys that store information about the views and
settings of folders. These keys record details such as folder display settings, icon sizes, window
positions, and sort order. ShellBags keys are located under the following registry paths:
•	 USRCLASS.DAT\Local Settings\Software\Microsoft\Windows\Shell\Bags
•	 USRCLASS.DAT\Local Settings\Software\Microsoft\Windows\Shell\BagMRU
•	 SOFTWARE\Microsoft\Windows\Shell\Bags
•	 SOFTWARE\Microsoft\Windows\Shell\BagMRU
When a user opens and interacts with a folder, the operating system creates or updates entries
within the ShellBags keys. These entries store the folder's unique identifier, known as its
Shell ID, along with its configuration. 4Discovery reviewed the ShellBags to determine whether
there was evidence of the Production Files.
WordWheelQuery
The WordWheelQuery key is a registry key that stores a history of searches made through
Windows Explorer. The WordWheelQuery key is located in the following locations in the
SOFTWARE hive:
•	 SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery
4Discovery reviewed the WordWheelQuery entries to determine whether there was
evidence of searches relating to the Production Files.
