  • Wireshark: The screenshots from Wireshark are crucial. Wireshark captures all network traffic. The "Info" column often provides clues about the protocol (HTTP, HTTPS, TLS, etc.) and the nature of the communication. The presence of unencrypted HTTP traffic (as opposed to HTTPS) would be a significant security concern. Test Description: The test objective centers on identifying potential vulnerabilities in server responses to non-HTTPS traffic. Encryption through HTTPS is a standard security practice.   • Hexdumps: The hexdumps represent raw data. Without specialized tools and knowledge of the specific protocols being used, it's very difficult to interpret them directly. However, they provide a complete record of the data exchanged.
