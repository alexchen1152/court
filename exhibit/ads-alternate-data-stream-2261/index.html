ADS is a feature of the Microsoft Windows NT file system (‚ÄúNTFS‚Äù) that allows a user to attach data to a file but keep that data separate from the main file. This allows a user to have multiple ‚Äústreams‚Äù of information in one file. Importantly, data stored in an ADS is hidden and not viewable when using standard file browsing methods, such as Windows Explorer. Storing data in an ADS can be used for many purposes, such as compatibility with other operating systems, but is also used by malicious actors who seek to obfuscate their activities from detection.
